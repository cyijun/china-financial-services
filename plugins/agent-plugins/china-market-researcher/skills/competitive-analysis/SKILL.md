# 中国A股市场竞争格局分析

构建完整的竞争分析材料。这是一个两阶段流程：先收集需求并获得大纲确认，再实际构建。

## 环境检查

本技能在PowerPoint插件和对话中均可工作。开始前先确认当前环境 — 操作方式不同，工作流程一致：

- **插件** — 演示文稿已打开；直接在其中构建幻灯片。
- **对话** — 生成 `.pptx` 文件（或构建到用户上传的文件中）。

以下内容适用于两种环境。

---

## 第一阶段 — 界定分析范围

竞争分析对不同人意义不同。在开展任何研究或建幻灯片之前，先通过 `ask_user_question` 确认用户真正需要什么。不要猜测 — 20页的同业对标材料与5页的市场地图都叫"竞争分析"，但形态和工作量完全不同。

尽量在一轮中收集完毕（该工具最多支持4个问题）：

- **范围** — 以单一目标公司为中心、周边环绕竞争对手？还是多家公司并排对比、没有主角？
- **竞争对手集合** — 哪些公司在范围内？如果用户点名了，就精确使用。如果用户说"行业龙头"，先提出一个集合并确认。
- **受众与深度** — 给圈内人的快速阅读，还是给外行的全面入门？这决定是否需要市场规模、行业经济学和历史沿革 — 还是可以直接跳到对比。
- **投资语境** — 是否需要牛/中/熊情景和观测指标？那是下面的第9步；如果是战略回顾而非投资分析，可以跳过。

如果用户上传了包含竞争对手数据的Excel/CSV，确认哪些列对应哪些指标后再开始拉数。源文件保真度很重要：按给定值精确使用，不要重新计算或四舍五入。

---

## 第二阶段 — 大纲、确认、再构建

**在大纲获批之前不要创建幻灯片。** 提出幻灯片标题和一行内容摘要，展示给用户，获得确认。一份竞争材料通常10-20页相互关联的内容 — 因为第4页错了而返工代价高昂。大纲是便宜的迭代点。

在提出大纲时，`ask_user_question` 很适合做结构性决策：采用哪种定位可视化（2×2矩阵 / 雷达图 / 梯队图 — 见第5步）、如何分组竞争对手（按商业模式 / 细分 / 姿态 — 见第4步）。这些通常是口味问题，用户往往有明确偏好。

## 贯穿始终的标准

### 提示忠实度

当用户指定了某些内容时，那是要求，不是建议：
- **幻灯片标题和章节名** — 精确用词。如果用户说"概述与竞争范围"，不要换成"2024财年竞争格局"。
- **图表 vs 表格** — 不可互换。"嵌入图表"意味着真正的图表对象，数据标签在柱/饼上，不是格式化的文字表格。
- **完整数据系列** — 如果用户列出7个竞争对手，全部纳入。如果展示2015-2025，每年都要有。
- **精确值和比率** — "以4:1超越对手A，以8:1超越对手B" 意味着就是这些比率，不是"约7.6倍对手B"。

### 数据源质量与冲突处理

1. `tushare-data` skill 财务数据（`fina_indicator`、`income`、`stock_basic`） — 结构化、可追溯
2. 公司年报/定期报告（经审计）
3. 券商研报摘要 — 分析师估计，可用于非上市公司规模估算
4. 行业报告（McKinsey、艾瑞、易观、券商研究所 — 市场规模与趋势）
5. Web搜索 — 新闻与政策动态（仅近期发展；需与主要来源交叉验证）
6. 公司公告（`anns_d`）与投资者交流纪要

### 数据可比性

- 所有竞争对手指标取自同一财年；明确标注例外（"2024年报" vs "2024年中报"）
- 跨竞争对手使用相同的指标定义
- 涉及港股或ADR时使用一致汇率，注明汇率和日期
- 缺失数据显示为 "-" 或 "N/A"，并带 "[估]" 标记表示估计值 — 绝不留空
- 每个数字都有引用："[公司] [文件] ([日期])" 或 "Web搜索-[来源]"

### 设计

- **幻灯片标题是洞察，不是标签。** "规模龙头与细分选手的差距正在拉大" — 不是"竞争分析"。
- **观测指标是量化的。** "毛利率低于40%" — 不是"利润率下滑"。
- **评级展示实际值。** "●●● 1600亿" — 不只是"●●●"。
- **图表是真正的图表对象** — 不是打扮成图表的文字表格。

**排版** — 明确设置，不依赖默认值：
- 幻灯片标题：28-32pt 加粗
- 章节标题：18-20pt 加粗
- 正文：14-16pt（绝不小于14pt）
- 表格文字：14pt
- 来源/脚注：14pt，灰色
- 同类元素 = 全篇统一字号

**图表：**
- 图例在图表边界内，不要悬浮在绘图区上方
- 饼图右侧图例（≤6片），折线/柱状图底部图例（≤4系列）
- 超过6个系列 → 拆成多个图表或改用表格
- 饼图在切片上显示百分比，不只是图例

**表格：**
- 浅灰表头行，加粗
- 数字右对齐，文字左对齐
- 单元格内边距足够，文字不贴边

**颜色：** 最多2-3种。柔和 — 藏青、灰色、一种强调色。同一种颜色含义全篇一致。

### 严格 vs 灵活的界限

| 始终严格 | 视情况而定 |
|---|---|
| 用户指定的精确标题/章节 | 用户未指定时的创意标题 |
| 用户说图表就用图表；说表格就用表格 | 未指定时的可视化类型 |
| 用户列出的每个竞争对手/数据点 | 未指定时的竞争对手数量 |
| 用户指定的精确值 | 未指定精度时的取整 |
| 标题不溢出 | 竞争对手分类数量 |
| 元素不重叠 | 对比维度选择 |

## 分析工作流程

### 第0步 — 行业定义性指标

在一切之前：这个行业真正运转的3-5个指标是什么？在所有竞争对手间一致使用。

| 行业 | 关键指标 |
|---|---|
| 白酒 | 营收、毛利率、预收款/合同负债、产能、吨价 |
| 新能源/动力电池 | 出货量/装机量、毛利率、产能利用率、研发费用率 |
| 互联网平台/电商 | GMV、货币化率、获客成本、用户时长、MAU |
| 银行 | 总资产、净息差、不良率、ROE、拨备覆盖率 |
| 医药/创新药 | 研发投入、管线进度、核心品种收入、毛利率 |
| SaaS/软件 | 收入增长、毛利率、预收款/合同负债、研发费用率 |
| 半导体 | 营收、毛利率、研发投入占比、产能利用率 |

行业未列出 — 选择投资者和经营者用于对标的指标。

### 第1步 — 市场语境

规模、增速、驱动因素、逆风。附来源。

正确："中国动力电池装机量2024年约500GWh，同比增长约25%（中国汽车动力电池产业创新联盟2024）"
错误："市场很大且增长迅速"

通过 `tushare-data` skill 获取：
- `sw_daily` / `ths_index` — 申万/同花顺行业指数走势与估值
- `index_classify` — 行业分类与成分股
- Web搜索 — 行业TAM、政策文件、券商研报摘要

### 第2步 — 行业经济学

绘制价值如何流动。方法取决于行业结构：
- **垂直结构化** — 价值链层级，每层典型利润率
- **平台/网络** — 生态参与者，价值在他们之间如何流动
- **碎片化** — 整合动态，规模差异带来的利润率分化

中国市场的特殊考量：政策补贴/退坡、地方政府产业引导、国企vs民企竞争格局、供应链本土化（国产替代）。

### 第3步 — 目标公司画像

```
| 指标 | 数值 |
|---|---|
| 营业收入 | 496亿元 |
| 增速 | +26% YoY |
| 毛利率 | 45% |
| 盈利能力 | 37.3亿元 归母净利润 |
| 客户数 | 134万 |
| 留存率 | 92% |
| 市场份额 | ~15% |
```

多业务公司增加分部拆解：

```
| 分部 | 营收 | 营收YoY | 营收占比 | 净利润 | 净利润YoY | 净利率 |
|---|---|---|---|---|---|---|
| 分部A | 251亿 | +26% | 57% | 65亿 | +31% | 26% |
| 分部B | 138亿 | +31% | 31% | 25亿 | +64% | 18% |
| 分部C | 51亿 | -2% | 12% | -7.4亿 | -16% | -1% |
| 合计 | 440亿 | +18% | 100% | 65亿* | - | 15% |
```
*如有总部费用请注明

数据来源：`tushare-data` skill 调用 `income`、`fina_indicator`；Web搜索补充非财务指标。

### 第4步 — 竞争对手映射

按最适用的视角分组（如果用户未指定，这是 `ask_user_question` 的好决策点）：
- 按商业模式 — 平台 / 垂直 / 水平
- 按细分 — 企业 / SMB / 消费者
- 按姿态 — 直接 / 邻近 / 新兴
- 按出身 — 传统龙头 / 颠覆者 / 新进入者

### 第5步 — 定位可视化

| 类型 | 适用场景 |
|---|---|
| 2×2矩阵 | 两个主导竞争因素 |
| 雷达/蜘蛛图 | 多因素对比 |
| 梯队图 | 自然聚类为战略群体 |
| 价值链地图 | 垂直行业 |
| 生态地图 | 平台市场 |

中国市场示例：白酒行业可按"价格带（高端/次高端/大众）"和"全国化程度"做2×2；新能源可按"技术路线"和"装机规模"做2×2。参见 `references/frameworks.md` 获取更多行业 2×2 轴对示例。

### 第6步 — 竞争对手深度剖析

每个竞争对手两张表。

**量化指标：**
```
| 指标 | 数值 |
|---|---|
| 营业收入 | XX亿元 |
| 增速 | +XX% YoY |
| 毛利率 | XX% |
| 总市值 | XX亿元 |
| 盈利能力 | XX亿元 归母净利润 |
| 市场份额 | ~XX% |
```

**定性：**
```
| 维度 | 评估 |
|---|---|
| 业务 | 一句话概括其业务 |
| 优势 | 2-3条 |
| 劣势 | 2-3条 |
| 战略 | 当前重点方向 |
| 最新动态 | 近期公告/并购/政策影响 |
```

数据来源：`tushare-data` skill 调用 `stock_basic`、`fina_indicator`、`income` 获取财务指标；Web搜索获取市场份额、战略动态、管理层变动；`tushare-data` skill 调用 `anns_d` 获取官方公告。

### 第7步 — 对比分析

```
| 维度 | 公司A | 公司B | 公司C |
|---|---|---|---|
| 规模 | ●●● 1600亿 | ●●○ 450亿 | ●○○ 80亿 |
| 增速 | ●●○ +26% | ●●● +35% | ●●○ +22% |
| 利润率 | ●●○ 7.5% | ●○○ 3.2% | ●●● 15% |
| 估值(PE) | ●●○ 25x | ●○○ 12x | ●●● 45x |
```

### 第8步 — 战略语境

**并购动态** — 中国A股市场的并购需关注：监管审批（证监会/交易所）、反垄断审查、借壳上市限制、跨界并购监管态度。

**融资模式** — A股IPO、定增、可转债、配股的近期趋势；北交所/科创板/创业板的差异化定位。

**政策与监管** — 产业政策（如新能源补贴退坡、半导体大基金、集采扩围）、监管变化（如反垄断、数据安全、境外上市备案）。

**资本流动** — 北向资金（`moneyflow_hsgt`）、主力资金、板块资金流向。

数据来源：Web搜索（政策文件、监管公告）；`tushare-data` skill 调用 `moneyflow_hsgt`、`anns_d`。参见 `references/schemas.md` 获取并购交易表和情景分析表的标准格式。

### 第9步 — 综合研判

**护城河评估** — 按以下维度对每个竞争对手评级（强 / 中等 / 弱）：

| 护城河 | 评估要点 |
|---|---|
| 网络效应 | 用户/供应商飞轮强度；跨边 vs 同边 |
| 转换成本 | 技术集成深度、合同锁定、行为习惯 |
| 规模经济 | 规模化的单位成本优势；最小有效规模 |
| 无形资产 | 品牌、专有数据、监管牌照、专利/技术秘密 |
| 成本优势 | 资源禀赋、地理位置、供应链整合 |
| 政策壁垒 | 准入许可、国资背景、特许经营 |

**必需的综合元素：**
- 持久优势（难以复制）— 映射到护城河类别
- 结构性弱点（难以修复）
- 当前状态 vs 趋势轨迹

**投资语境下的情景分析**（如果第一阶段界定中用户需要）：

```
| 情景 | 概率 | 关键驱动 |
|---|---|---|
| 牛市 | 30% | 市场份额提升、利润率扩张、政策超预期 |
| 基准 | 50% | 当前轨迹延续 |
| 熊市 | 20% | 竞争加剧、价格战、政策收紧、需求不及预期 |
```

中国市场特征：政策影响权重高、行业周期性强（新能源/半导体/猪周期等）、散户情绪对估值波动影响大、国企背景在某些行业是重要变量。

## 质量检查清单

完成前检查：

**提示忠实度**
- 幻灯片标题与用户指定的完全一致，逐字
- 用户说图表就用图表；说表格就用表格
- 用户列出的每个竞争对手/年份/数据点都已呈现
- 值和格式与用户指定的一致

**数据一致性**
- 源文件值直接提取，未重新计算
- 同一指标在每张幻灯片上显示相同值
- 与源文件相同的小数精度

**布局**
- 标题不溢出
- 元素不重叠
- 所有文字在容器内，无截断

**内容**
- 每个数字都有引用
- 所有指标来自同一财年（或已标注）
- 幻灯片标题陈述洞察，不是话题
- 图表是真正的图表对象

对每张幻灯片运行标准视觉验证检查 — 这能发现重叠、溢出和低对比度文字，这些问题在读XML时不会显现。

## 引用格式规范

**Tushare数据：** "Tushare `fina_indicator` 接口，[公司] 2024年报，ROE"
**公司年报：** "[贵州茅台] 2024年报，合并利润表"
**券商研报：** "[中金公司/中信证券] 研报 [标题]，2024-10"
**Web搜索：** "Web搜索-[艾瑞咨询/易观/McKinsey/行业新闻]，[标题/主题]，[日期]"
**政策文件：** "[国务院/工信部/证监会] [文件名]，[日期]"
**公告：** "[公司] `anns_d` 公告，[公告类型]，[日期]"

当多个来源冲突时，优先采用：Tushare/交易所公告 > 公司投资者材料 > 券商研报 > 行业报告 > 新闻报道。
