# 行业概览 (Sector Overview)

为指定A股行业（申万/中信分类或自定义主题）生成一份结构化的深度概览报告，融合量化数据与定性研究，支持投资研究、客户汇报、竞争分析及M&A评估等场景。

## 数据源

- **主数据源**：`tushare-data` skill（自动处理 `tushare` Python 包及 `TUSHARE_TOKEN` 认证）
- **辅助数据源**：网络搜索（券商研报、艾瑞/易观等咨询报告、政府产业规划、新闻资讯）
- **市场范围**：A股为主，可选含港股通/美股中概股作为对照

## `tushare-data` skill 核心接口

| 目的 | 接口 |
|------|------|
| 股票列表与行业分类 | `stock_basic`, `index_classify` |
| 行业指数行情 | `sw_daily`, `index_daily` |
| 概念/主题指数 | `ths_index`, `ths_member` |
| 个股估值 | `daily_basic` |
| 财务指标 | `fina_indicator` |
| 财务报表 | `income`, `balancesheet`, `cashflow` |
| 资金流向 | `moneyflow`, `moneyflow_hsgt` |
| 宏观数据 | `cn_cpi`, `cn_ppi`, `cn_pmi`, `cn_m`, `sf_month` |
| 跨市场数据 | `hk_daily`, `us_daily`, `index_global` |
| 研报与公告 | `research_report`, `anns_d` |

## Step 2：市场概览 (Market Overview)

### 2.1 市场规模 (TAM)

- **国内市场规模**：通过网络搜索获取券商研报、艾瑞/易观/前瞻产业研究院、政府统计局数据
- **全球市场规模及中国占比**：评估中国企业在全球产业链中的地位
- **细分市场**：按产品类型、地域（华东/华南/中西部）、终端应用（B端/C端/G端）拆分
- **产业链价值分布**：上游（原材料/设备）、中游（制造/组装）、下游（品牌/渠道）各环节利润率对比

### 2.2 历史增速

- 通过 `tushare-data` skill 调用 `sw_daily` 或 `index_daily` 获取行业指数过去5年的收益数据
- 计算行业指数的年化收益率、波动率、最大回撤
- 与沪深300、中证500对比，计算超额收益（Alpha）
- 分年度表现：识别行业的周期性特征（如强周期、弱周期、成长型）

```python
# 示例：获取申万行业指数历史数据
df = pro.sw_daily(ts_code='801XXX.SI', start_date='20190101', end_date='20241231')
# 计算年化收益、波动率
```

### 2.3 预测增速

- 通过网络搜索获取券商一致预期（Wind/朝阳永续汇总）
- 结合宏观数据（`cn_pmi`, `cn_ppi`, `sf_month`）判断行业景气度方向
- 政府产业规划中的目标增速（如"十四五"规划、各部委产业政策）

### 2.4 行业结构

- **集中度**：CR3 / CR5 / CR10（基于上市公司营收数据从 `fina_indicator` 计算）
- **价值链分析**：各环节利润率、资本密集度、技术壁垒
- **商业模式**：TO B（合同/订单制）vs TO C（零售/平台）vs TO G（招投标）
- **进入壁垒**：牌照、技术专利、资本、渠道、品牌
- **替代品威胁**：跨界颠覆风险（如新能源对传统能源、电商对零售）

### 2.5 关键趋势

通过网络搜索与 `tushare-data` skill 获取的数据结合分析：

- **长期顺风（Secular Tailwinds）**：
  - 国产替代（半导体、工业软件、医疗器械）
  - 能源转型（光伏、储能、新能源车、电网升级）
  - 人口结构变化（老龄化→医药/银发经济；少子化→宠物/ automation）
  - 消费升级/降级并存
  
- **逆风与风险（Headwinds）**：
  - 产能过剩与价格战（光伏、锂电、钢铁）
  - 监管政策变化（教培、互联网、医药集采、地产）
  - 技术路线不确定性（如锂电vs固态电池、PERC vs HJT vs TOPCon）
  - 国际贸易摩擦（关税、出口管制）

- **技术颠覆**：
  - AI对行业的赋能或替代效应
  - 新工艺/新材料对成本曲线的影响

- **并购活动**：
  - 近期A股同行业并购案例（通过网络搜索）
  - 产业资本 vs 财务资本的投资逻辑差异

## Step 3：竞争格局 (Competitive Landscape)

### 3.1 头部企业画像

筛选行业前5-10家上市公司，生成对比表：

```python
# 获取行业内所有股票列表
# 通过 stock_basic 筛选行业代码
# 通过 fina_indicator 获取最新财报数据
```

**对比表结构**：

| 公司 | 股票代码 | 总市值(亿) | 营收(亿) | 营收增速 | 毛利率 | 净利率 | ROE | 资产负债率 | 研发费用率 | 市场份额 | 核心差异点 |
|------|----------|-----------|----------|----------|--------|--------|-----|-----------|-----------|----------|----------|
| 公司A | 6XXXXX.SH | XXX | XX.X | XX% | XX% | XX% | XX% | XX% | XX% | X% | ... |
| 公司B | 0XXXXX.SZ | XXX | XX.X | XX% | XX% | XX% | XX% | XX% | XX% | X% | ... |

**数据来源**：
- 市值：`daily_basic`
- 营收/增速/毛利率/净利率/ROE：`fina_indicator`
- 资产负债率：`fina_indicator` 或 `balancesheet`
- 研发费用率：`income` 中 research_expenses / total_revenue
- 市场份额：基于上市公司营收加总估算，或通过**网络搜索**获取第三方数据

### 3.2 竞争动态分析

- **价格战**：是否处于价格下行周期？哪些子领域最激烈？
- **产品竞争**：技术代差、产品迭代速度、SKU丰富度
- **渠道竞争**：线上vs线下、国内vs海外、直销vs经销
- **服务竞争**：交付周期、售后、定制化能力
- **份额变化**：近3年各公司营收增速差异反映的份额迁移
- **新进入者威胁**：跨界玩家（如互联网巨头进入造车、家电）
- **退出壁垒**：资产专用性、员工安置、政策限制

### 3.3 护城河评估

对头部企业逐一评估：
- **成本优势**：规模效应、资源禀赋、工艺know-how
- **品牌溢价**：消费者心智占有率、定价权
- **网络效应**：平台型企业的双边网络
- **转换成本**：B端客户的迁移成本
- **牌照/特许经营权**：金融、公用事业、通信等

## Step 4：估值语境 (Valuation Context)

### 4.1 行业估值水位

- 通过 `tushare-data` skill 调用 `daily_basic` 获取行业内所有成分股的PE-TTM、PB、PS-TTM
- 计算行业估值分布（中位数、均值、10%/90%分位）
- 绘制行业PE/PB历史区间（近5年），标注当前所处百分位

```python
# 示例：计算行业PE分布
pe_list = []
for ts_code in sector_stocks:
    df = pro.daily_basic(ts_code=ts_code, trade_date='YYYYMMDD')
    pe_list.append(df['pe_ttm'].values[0])
# 计算中位数、均值、分位数
```

### 4.2 溢价/折价驱动因素

- **估值溢价原因**：高成长确定性、强护城河、稀缺性（A股唯一标的）、北向资金偏好
- **估值折价原因**：周期性、政策风险、治理问题、流动性差、ESG争议
- **中外估值对比**：A股 vs 港股通标的 vs 美股中概的估值差异及原因

### 4.3 近期A股并购交易估值

- 通过网络搜索获取近1-2年同行业A股并购案例
- 记录交易估值倍数（PE、PB、EV/EBITDA）
- 与二级市场估值对比，评估是否存在并购溢价/折价

### 4.4 相对大盘估值

- 行业PE / 沪深300 PE → 相对估值溢价/折价
- 行业PE / 中证500 PE → 相对中盘估值
- 历史相对估值区间：当前处于历史中高位还是中低位

## A股特别注意事项

### 区分TAM hype vs 真实市场
- A股常见"画大饼"现象：概念股描述的TAM极大，但实际可触达市场（SAM/SOM）很小
- 务必通过网络搜索验证：政府规划中的目标是否落地？产能规划是否已过剩？
- 案例：某"AI+医疗"概念公司，TAM万亿，但A股产品收入占比不足5%

### 数据时效性
- A股财报披露有季节性（1-4月年报、4-8月中报、10月三季报）
- 务必标注所有数据的截止日期，避免使用过时的财务数据做决策
- 行业指数数据建议使用最新交易日；财务数据使用最新完整报告期

### 政策驱动行业的监管风险
- 对于强政策驱动行业（新能源、医药、教培、互联网、地产、金融），需单独设置"政策风险"章节
- 通过网络搜索持续跟踪：监管态度变化、政策文件征求意见稿、行业整顿动态
- 政策转向可能是最大Alpha来源，也可能是最大风险来源

### 图表建议
- **行业PE-band**：最能体现A股行业估值历史位置，是买方汇报的核心图表
- **成分股涨幅分布**：识别行业内的alpha来源（是龙头集中还是分散普涨）
- **估值散点图（PE vs 增速）**：快速识别便宜的高增长标的（左下角区域）
- **北向资金+主力资金流向图**：验证外资与内资对行业的态度是否一致

### 客户定制场景
- **M&A目标筛选**：增加控制权溢价分析、协同效应评估、反垄断审查风险
- **竞争定位**：增加与全球龙头的对标分析（通过网络搜索获取海外公司数据）
- **市场进入咨询**：增加渠道成本、客户获取成本、监管准入门槛分析

