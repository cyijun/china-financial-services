# 投资点子筛选 (Idea Generation)

系统性地从A股全市场生成5-10个具有数据支撑的投资点子，结合量化筛选与主题研究，适配不同风格（价值/成长/质量/主题/事件驱动）。

## 数据源

- **主数据源**：`tushare-data` skill（自动处理 `tushare` Python 包及 `TUSHARE_TOKEN` 认证）
- **辅助数据源**：网络搜索（行业研报、政策文件、新闻资讯、券商策略）
- **市场范围**：以A股为主，可含港股通标的

## `tushare-data` skill 核心接口

| 目的 | 接口 |
|------|------|
| 股票列表与基本信息 | `stock_basic` |
| 每日估值指标 | `daily_basic` |
| 财务指标（ROE、毛利率、增速等） | `fina_indicator` |
| 资金流向（个股/板块） | `moneyflow` |
| 北向资金（沪深港通） | `moneyflow_hsgt` |
| 龙虎榜 | `top_list`, `top_inst` |
| 涨停板/连板数据 | `limit_list_d`, `limit_step` |
| 热点题材 | `dc_hot`, `ths_hot` |
| 行业/概念指数成分 | `index_classify`, `ths_member` |
| 公告与研报 | `anns_d`, `research_report` |
| 财务三张表（深度验证） | `income`, `balancesheet`, `cashflow` |

## Step 1：明确筛选维度

在开始筛选前，先与使用者确认以下维度：

| 维度 | 选项 |
|------|------|
| **方向** | 看多 / 看空 / 多空对冲 |
| **市值** | 大盘 (>500亿) / 中盘 (100-500亿) / 小盘 (20-100亿) / 微盘 (<20亿) |
| **行业** | 申万/中信一级或二级行业，或不限 |
| **风格** | 价值 / 成长 / 质量 / 主题 / 事件驱动 |
| **地域** | A股为主 / 含港股通 |
| **主题** | AI算力、新能源、消费复苏、红利高息、国产替代、军工、医药创新等 |

如使用者未指定，默认执行**全市场多风格扫描**，覆盖价值、成长、质量各2-3个，主题/事件1-2个。

---

## Step 2：量化筛选

### 2.1 价值筛选 (Value Screen)

目标：寻找估值低于内在价值、现金流稳健、分红意愿强的标的。

通过 `tushare-data` skill 获取：调用 `daily_basic`（最新交易日估值指标）结合 `fina_indicator`（最新财报数据）。

**筛选条件**：
- PE-TTM < 所在申万二级行业中位数
- PB < 自身近3年历史均值（或行业中位数）
- 股息率 > 3%（红利风格可放宽至>2%）
- 市现率（股价/经营现金流）合理，< 行业中位数
- 资产负债率 < 行业中位数（金融地产除外）
- 经营现金流连续2年为正

**辅助验证**：
- `fina_indicator` 中 `ocfps`（每股经营现金流）稳定性
- 近3年分红率是否稳定或提升
- 是否存在大额商誉减值风险（查 `balancesheet` 中 goodwill）

### 2.2 成长筛选 (Growth Screen)

目标：寻找高增长、高盈利能力、研发驱动的标的。

**筛选条件**：
- 营业收入同比增速 > 20%（近2季度或TTM）
- 净利润同比增速 > 25%（扣非后优先）
- 毛利率同比扩张或维持高位
- ROE > 15%（或ROE持续提升）
- 研发费用/营业收入比持续提升（科技成长类优先）
- 预收账款/合同负债增长（`balancesheet`）

**辅助验证**：
- 增速是否来自主业（扣非 vs 归母净利润差异）
- 现金流是否匹配利润增长（经营现金流/净利润 > 0.8）
- 增速可持续性：订单、产能、客户集中度

### 2.3 质量筛选 (Quality Screen)

目标：寻找商业模式优秀、财务稳健、治理良好的长期持有标的。

**筛选条件**：
- 连续5年营业收入正增长
- 毛利率稳定或扩张（波动率小）
- ROE > 15% 且稳定
- 资产负债率 < 行业中位数（或低于50%）
- 经营现金流净额 ≥ 净利润（连续3年）
- 大股东持股比例稳定（查 `top10_holders` 或公告）
- 审计意见为标准无保留（`fina_audit`）

**辅助验证**：
- 应收账款周转天数、存货周转天数趋势
- 资本开支/折旧摊销比（判断再投资效率）
- 有息负债率变化趋势

### 2.4 做空/回避筛选 (Short / Avoid Screen)

目标：识别基本面恶化、财务风险高、估值泡沫的标的，用于回避或配对做空。

**筛选条件**：
- 营业收入同比下滑或增速连续放缓
- 毛利率同比压缩
- 应收账款增速 > 营收增速，或存货增速 > 营收增速
- 大股东/高管持续减持（查 `anns_d` 中减持公告）
- 估值（PE/PB）高于同业均值但缺乏基本面支撑
- 股权质押比例高（>50%）
- 审计意见非标（保留、无法表示、否定）
- 经营现金流净额连续为负或大幅低于净利润
- 商誉/净资产 > 30%

**辅助验证**：
- `balancesheet` 中短期借款+一年内到期非流动负债 vs 货币资金
- `cashflow` 中筹资现金流持续大额流出（偿债压力）
- 北向资金持续净流出

### 2.5 主题/事件筛选 (Thematic / Event Screen)

目标：捕捉市场热点、政策催化、产业拐点的交易性机会。

**筛选条件**：
- 连板梯队结构分析（`limit_list_d`, `limit_step`）：识别龙头、跟风、补涨标的
- 热点题材追踪（`dc_hot`, `ths_hot`）：当日/近5日最热门概念
- 政策催化：通过**网络搜索**获取最新政策（如工信部、发改委、证监会文件）
- 行业拐点信号：价格触底回升、库存周期切换、产能利用率变化

**辅助验证**：
- 主题纯正度：通过 `ths_member` 或 `index_classify` 确认标的与主题的相关性
- 资金流向：主题内资金是集中龙头还是扩散炒作
- 事件确定性：业绩发布、订单公告、重组进展（`anns_d`）

### 2.6 北向/主力资金筛选 (Capital Flow Screen)

目标：跟踪聪明钱动向，验证或增强其他筛选信号。

**筛选条件**：
- 北向资金连续5日/10日净流入（`moneyflow_hsgt`）
- 个股主力资金持续净流入（`moneyflow`，大单+特大单）
- 龙虎榜机构席位净买入（`top_list`, `top_inst`）
- 北向资金持股比例变化（近1月/3月）

**辅助验证**：
- 北向流入是否伴随股价上涨（量价配合）
- 机构龙虎榜买入后5日/20日胜率统计
- 主力资金流向与北向是否一致（共振信号更强）

## Step 3：主题深度扫描 (Thematic Sweep)

对Step 2筛选出的主题类标的，进行深度定性研究：

1. **产业链映射**
   - 使用网络搜索获取产业链图谱（上游、中游、下游）
   - 通过 `tushare-data` skill 调用 `ths_member` 或 `index_classify` 确认各环节的A股标的
   - 判断当前市场炒作的是产业链的哪一环节

2. **纯正度 vs 多元化**
   - 纯正度：营收中主题相关业务占比（通过 `income` 按产品拆分或研报确认）
   - 多元化：是否为跨界蹭热点，主业与主题关联度低

3. **预期差识别**
   - 已充分定价（Price-in）：股价已大幅上涨，机构覆盖度高，预期一致
   - 未被充分认知（Under-appreciated）：基本面改善但股价未反应，或存在信息差
   - 通过**网络搜索**券商覆盖度、分析师预期、机构持仓变化辅助判断

4. **政策与监管风险评估**
   - 通过**网络搜索**最新监管动态（如反垄断、数据安全、环保限产）
   - 评估政策对主题的催化或压制作用

## Step 5：最终输出

输出一份结构化的投资点子报告：

1. **筛选条件说明**：记录本次使用的具体筛选参数（市值范围、行业、财务阈值等）
2. **点子清单**：5-10个投资点子，每个包含点子卡片
3. **对比总表**：所有点子核心指标横向对比
4. **优先级排序**：
   - 高优先级：多维度共振（如价值+北向流入+政策催化）
   - 中优先级：单一维度突出但存在不确定性
   - 低优先级/观察：需要进一步验证
5. **附录**：
   - 数据日期标注
   - 使用的 `tushare-data` skill 接口列表
   - 网络搜索关键信息来源

## A股特别注意事项

### 避免过度拥挤交易
- **检查北向持仓**：北向资金持仓比例过高的标的，边际买入空间可能有限
- **检查机构持仓**：基金重仓股需评估是否存在赎回导致的被动卖出风险
- **分析师覆盖度**：过度覆盖的标的往往预期充分，超额收益空间有限；低覆盖标的可能存在信息差

### 主题投资注意政策风险
- A股主题投资高度依赖政策导向，需持续跟踪监管态度
- 警惕"一日游"题材：通过 `limit_list_d` 观察连板持续性，1-2板后断板频繁的题材谨慎参与
- 政策敏感行业（教培、互联网、医药集采、地产）需额外评估监管风险

### 小盘股注意流动性风险
- 微盘/小盘股（<50亿）日均成交额需>5000万方可考虑
- 注意大股东解禁时间表（`anns_d` 中限售股解禁公告）
- 股权质押比例高的标的，下跌时可能触发平仓螺旋

### 跟踪筛选胜率
- 建议记录每次筛选结果，回测1个月/3个月后的表现
- 统计各筛选策略（价值/成长/质量/主题）在A股市场的实际胜率
- 持续优化阈值：A股市场的有效阈值可能与美股不同（如PE<15在A股可能过于严格）

## 示例工作流

```
使用者：帮我找找A股当前有什么好的投资机会，偏向价值和质量风格。

Agent：
1. 调用 pro.daily_basic(trade_date=最新交易日) 获取全市场估值数据
2. 调用 pro.fina_indicator(ann_date=最新财报期) 获取财务指标
3. 执行价值筛选：PE-TTM < 行业中位数，股息率 > 3%，PB < 历史均值
4. 执行质量筛选：ROE > 15%，经营现金流 > 净利润，资产负债率 < 50%
5. 取交集，得到候选池
6. 对候选池调用 moneyflow_hsgt 和 moneyflow 验证资金动向
7. 对前10名通过网络搜索获取最新基本面变化和催化剂
8. 输出5-8个投资点子卡片及对比总表
```
