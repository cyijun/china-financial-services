# 首次覆盖（A股）

通过结构化的5个任务工作流创建机构级A股首次覆盖研报。每个任务必须单独执行并验证输入。

## 概述

本技能产出综合性首次覆盖报告，遵循国内头部券商标准格式（中金公司、中信证券、华泰证券、国泰君安）。任务逐一执行，每一步都验证前置条件后再继续。

**默认字体**: 全文档使用宋体/SimSun（除非用户另有指定）。

## 任务1：公司研究

**目的**：研究公司业务、管理层、竞争地位、行业和风险。

**前置条件**：✅ 无（完全独立）
- 公司名称或股票代码（6位数字代码，如300750.SZ）

**流程**：
1. 验证公司名称/代码已提供
2. 从references/task1-company-research.md加载详细指令
3. 执行定性研究工作流
4. 交付研究文档

**产出**：公司研究文档（6,000-8,000字）
- 公司概述与历史
- 管理层简介（3-4位高管，每人300-400字）
- 产品与服务分析
- 行业概述
- 竞争分析（5-10家竞争对手）
- 市场规模测算（TAM）
- 风险评估（8-12项风险）

**文件名**：`[公司]_研究文档_[日期].md`

**⚠️ 只交付这1个文件。不要完成总结，不要额外文档。**

**⚠️ 不要走捷径：**
- ✅ 写满6,000-8,000字（不是摘要）
- ✅ 为所有3-4位高管完成300-400字简介
- ✅ 全面分析所有5-10家竞争对手
- ✅ 涵盖4个类别的全部8-12项风险
- ❌ 不要为了节省时间而缩写章节
- ❌ 不要跳过任何要求的章节

## 任务2：财务建模

**目的**：提取历史财务数据并构建全面的Excel财务模型，包含预测和情景分析。

**前置条件**：⚠️ 开始前验证
- **必需**：公司财务数据权限
  - 对于上市公司：最新年报/招股说明书，来源：中国证监会指定披露网站（巨潮资讯网 cninfo.com.cn）
  - 对于非上市公司：财务报表或可用估算
  - 或：用户提供的预提取历史财务数据
- **可选**：公司研究（任务1）以提供业务背景

**输入验证**：
```
开始前 - 选择方法：

选项A：提取财务数据（最常见）
- [ ] 是否有年报或财务报表权限？
- [ ] 准备提取3-5年历史数据？

选项B：用户提供预提取财务数据
- [ ] 是否收到历史财务数据文件？
- [ ] 是否包含利润表、现金流量表、资产负债表（3-5年）？

可选：
- [ ] 公司研究（任务1）是否已完成以提供背景？
```

**流程**：
1. 验证财务数据权限
2. 从references/task2-financial-modeling.md加载详细指令
3. **步骤1**：提取历史财务数据（如需要）
4. **步骤2+**：构建包含6个必备标签页的预测模型
5. 交付Excel模型

**产出**：Excel财务模型（.xlsx）
- 6个必备标签页：
  1. **收入模型** — 产品细分（20-30行）+ 地区细分（15-20行）
  2. **利润表** — 完整P&L，40-50行项目，历史（3-5年）+ 预测（5年）
  3. **现金流量表** — 经营/投资/筹资活动，历史 + 预测
  4. **资产负债表** — 资产/负债/权益，历史 + 预测
  5. **情景分析** — 乐观/基准/悲观对比表
  6. **DCF输入** — 为任务3估值做准备

**文件名**：`[公司]_财务模型_[日期].xlsx`

**⚠️ 只交付这1个文件。不要完成总结，不要额外文档。**

**⚠️ 不要走捷径：**
- ✅ 如需提取财务数据：从3张财务报表中提取所有行项目（3-5年）
- ✅ 完整构建所有6个预测标签页

- ✅ 构建包含40-50行项目的完整利润表（非缩写版）
- ✅ 包含完整的现金流量表和资产负债表及所有行项目
- ✅ 完成所有三种情景（乐观/基准/悲观）及不同参数
- ❌ 不要创建简化/缩写版本
- ❌ 不要跳过任何6个必备标签页
- ❌ 如需要，不要跳过历史财务数据提取

**继续到任务3前的验证**：
- [ ] 历史财务数据已提取（如需要）或已提供

## 估值方法论（A股适用）

### 1. DCF估值
- **自由现金流**：无杠杆自由现金流（UFCF）
- **WACC**：基于A股市场风险溢价（通常6%-8%）
- **终值**：Gordon增长模型或退出倍数法
- **敏感性**：WACC ±1%、终值增长率 ±0.5%

### 2. 可比公司法
- **PE法**：基于Tushare获取的市盈率数据
- **PB法**：适用于银行、保险等重资产行业
- **PS法**：适用于高成长、暂未盈利公司
- **EV-EBITDA**：适用于资本密集型行业
- **统计摘要**：最大值/75分位/中位数/25分位/最小值

### 3. 历史估值区间
- 使用Tushare获取近3-5年PE/PB Band
- 结合市场情绪判断当前估值位置

### 4. 目标价计算
- 综合DCF和可比公司法结果
- 通常采用多种方法的加权平均
- 目标价 = 当前价 × (1 + 预期收益率)

```
- 所有财务数据均基于中国企业会计准则（CAS）编制
- 历史数据来源于公司年报、半年报、季报及Tushare数据库
- 预测数据基于分析师假设，可能与实际结果存在重大差异
- 货币单位：人民币（CNY），除非另有说明
- 每股收益（EPS）按最新总股本计算
- 估值倍数基于交易日收盘价计算
```
