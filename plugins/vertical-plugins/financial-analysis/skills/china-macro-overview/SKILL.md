# 中国宏观经济概览

## ⚠️ 关键：数据源优先级（请先阅读）

**严格遵循以下数据源优先级：**

1. **首选：Tushare宏观数据接口** — 使用 `cn_cpi`、`cn_ppi`、`cn_pmi`、`cn_gdp`、`cn_m`、`shibor_lpr`、`sf_month` 等接口
2. **次选：中国人民银行、国家统计局官方发布** — CPI、PPI、GDP、货币供应量等官方数据
3. **备选：Wind/同花顺iFinD/Choice** — 若Tushare数据不全或需要更长历史序列
4. **参考：政策文件与央行报告** — 货币政策执行报告、政府工作报告、中央经济工作会议通稿

**避免使用通用网页搜索作为主要数据源** — 宏观数据要求官方权威来源，确保准确性与一致性。

---

## 概述

本技能指导Claude构建系统化的中国宏观经济监控框架，跟踪四大核心维度（通胀、增长、货币、利率），识别经济周期位置与趋势变化，输出结构化的宏观简报。简报设计为"一句话结论 + 关键数据 + 趋势判断"，便于快速决策参考。

**核心哲学：**
**"抓大放小，趋势优先。"**

宏观分析不需要穷尽所有指标，而是聚焦于对资产价格和投资决策最具影响力的核心变量，判断当前经济处于周期的哪个阶段，以及政策可能的应对方向。

## 监控框架：四大维度

### 维度一：通胀（Inflation）

**核心指标：**
- **CPI同比（居民消费价格指数）** — Tushare `cn_cpi`
  - 关注食品 vs 非食品分项
  - 关注核心CPI（剔除食品和能源）
  - 合理区间：1%-3%（温和通胀）；<1%（通缩压力）；>3%（通胀担忧）
- **PPI同比（工业生产者出厂价格指数）** — Tushare `cn_ppi`
  - 关注生产资料 vs 生活资料分项
  - PPI向CPI的传导（通常6-12个月滞后）
  - 负PPI意味着工业企业利润空间受压

**解读框架：**
```
CPI高 + PPI高 = 全面通胀 → 紧缩风险 → 债券利空、股市结构性机会
CPI高 + PPI低 = 消费通胀、工业通缩 → 滞胀迹象 → 防御配置
CPI低 + PPI高 = 上游通胀未传导 → 中游利润受压 → 关注传导能力
CPI低 + PPI低 = 双通缩 → 宽松预期 → 债券利好、股市待政策催化
```

### 维度二：增长（Growth）

**核心指标：**
- **GDP同比（国内生产总值）** — Tushare `cn_gdp`
  - 实际GDP vs 名义GDP
  - 季度增速与年度目标比对（如"5%左右"）
  - 三大产业贡献拆分
- **制造业PMI** — Tushare `cn_pmi`
  - 荣枯线50：>50扩张，<50收缩
  - 关注新订单、生产、就业、原材料库存分项
  - 官方PMI vs 财新PMI（样本差异：大企业 vs 中小企业）
- **非制造业PMI/服务业PMI** — Tushare `cn_pmi`
  - 建筑业PMI反映基建和房地产活动
  - 服务业PMI反映消费和商务活动

**解读框架：**
```
GDP > 目标 + PMI > 50 = 过热或偏热 → 政策收紧风险
GDP ≈ 目标 + PMI ≈ 50 = 平稳运行 → 政策中性
GDP < 目标 + PMI < 50 = 下行压力 → 宽松/刺激预期
```

### 维度三：货币（Monetary）

**核心指标：**
- **M2同比（广义货币供应量）** — Tushare `cn_m`
  - 反映整体流动性环境
  - 与名义GDP增速对比（M2-GDP gap 过大意味着流动性过剩）
- **社融增量/存量同比（社会融资规模）** — Tushare `sf_month`
  - 实体经济融资需求的晴雨表
  - 关注人民币贷款、政府债券、企业债券、表外融资分项
  - 社融增速与M2增速的剪刀差（反映资金空转或信用扩张）
- **M1同比** — Tushare `cn_m`
  - M1-M2剪刀差：差值扩大意味着企业活期存款增加、经营活跃度提升；差值收窄反之

**解读框架：**
```
M2高 + 社融高 = 宽货币+宽信用 → 流动性充裕 → 股债双牛环境
M2高 + 社融低 = 宽货币+紧信用 → 资金淤积 → 债市利好、股市观望
M2低 + 社融高 = 紧货币+宽信用 → 结构性支持 → 股市结构性机会
M2低 + 社融低 = 双紧 → 流动性收缩 → 防御为主
```

### 维度四：利率（Interest Rates）

**核心指标：**
- **LPR（贷款市场报价利率）** — Tushare `shibor_lpr`
  - 1年期LPR：实体经济贷款利率锚
  - 5年期LPR：房贷利率锚
  - 关注LPR下调频率与幅度（政策宽松信号）
- **SHIBOR（上海银行间同业拆放利率）** — Tushare `shibor_lpr`
  - 短期流动性松紧指标
  - 关注隔夜、7天、1月SHIBOR
- **10Y国债收益率** — Tushare `us_tycr` 或 `shibor_lpr`
  - 无风险利率基准
  - 与政策利率、经济增长预期的联动
- **期限利差（10Y-1Y国债）**
  - 利差扩大：经济扩张预期
  - 利差收窄/倒挂：经济衰退担忧

**解读框架：**
```
LPR下调 + 国债收益率下行 = 全面宽松 → 利好成长股、地产链
LPR下调 + 国债收益率上行 = 宽信用预期 → 利好周期股
LPR持平 + 国债收益率下行 = 避险情绪 → 利好债券、高股息
LPR持平 + 国债收益率上行 = 紧缩担忧 → 压制估值
```

## 经济周期定位

基于四大维度，将经济周期分为六个阶段：

| 阶段 | 增长 | 通胀 | 货币 | 利率 | 资产配置倾向 |
|------|------|------|------|------|-------------|
| 复苏早期 | 触底回升 | 低位 | 宽松 | 下行 | 股票 > 债券 > 现金 |
| 复苏中期 | 持续改善 | 温和回升 | 中性 | 企稳 | 股票 > 商品 > 债券 |
| 扩张后期 | 高位运行 | 升温 | 中性偏紧 | 上行 | 商品 > 股票 > 债券 |
| 滞胀 | 放缓 | 高位 | 收紧 | 高位 | 现金 > 商品 > 债券 |
| 衰退早期 | 快速下行 | 回落 | 开始宽松 | 下行 | 债券 > 现金 > 股票 |
| 衰退后期 | 筑底 | 低位 | 大幅宽松 | 低位 | 股票 > 债券 > 现金 |

**判断当前阶段的步骤：**
1. 看PMI/GDP趋势（方向比绝对值重要）
2. 看CPI/PPI趋势（通胀是上行还是下行）
3. 看社融/M2趋势（信用是扩张还是收缩）
4. 看LPR/国债收益率趋势（利率是下行还是上行）
5. 综合判断所处阶段及可能的演进方向

## 工作流程

### 第一步：数据采集

**必需数据（每次简报至少包含）：**
1. `cn_cpi` — 最新CPI同比及分项
2. `cn_ppi` — 最新PPI同比及分项
3. `cn_pmi` — 最新官方制造业PMI及分项
4. `cn_gdp` — 最新季度GDP同比（季度报告时）
5. `cn_m` — 最新M1、M2同比
6. `sf_month` — 最新社融增量及存量同比
7. `shibor_lpr` — 最新LPR、SHIBOR、国债收益率

**采集频率：**
- **周度简报**：PMI（月度初）、SHIBOR、国债收益率
- **月度简报**：CPI、PPI、M2、社融、LPR（若调整）
- **季度简报**：GDP及全面回顾

### 第二步：数据处理

1. **计算环比变化**：当前值 vs 上月/上季度/上年同期
2. **计算预期差**：实际值 vs 市场预期（如有）
3. **计算衍生指标**：
   - M1-M2剪刀差 = M1同比 - M2同比
   - 期限利差 = 10Y国债收益率 - 1Y国债收益率
   - 社融-M2增速差 = 社融存量同比 - M2同比
4. **标注异常值**：偏离历史均值2个标准差以上的数据

### 第三步：分析与判断

1. **逐项判断趋势**：对每个指标判断「上行/下行/走平」
2. **交叉验证**：不同维度之间是否一致（如PMI上行 + 社融扩张 = 复苏确认）
3. **周期定位**：根据趋势组合判断所处阶段
4. **政策推演**：基于当前状态推测政策应对方向

### 第四步：撰写简报

1. **先写一句话结论**：这是最重要的输出，要求精准、 actionable
2. **填充数据表格**：确保数据准确、对比清晰
3. **撰写趋势判断**：每个维度2-3句话，聚焦「方向」和「为什么」
4. **给出策略含义**：基于周期定位给出配置建议
5. **列出风险因素**：平衡视角，不遗漏主要风险

### 第五步：质量检查

- [ ] 所有数据已标注来源和日期
- [ ] 环比/同比变化方向正确
- [ ] 各维度判断之间逻辑自洽
- [ ] 一句话结论与数据支撑一致
- [ ] 策略含义与周期定位匹配
- [ ] 风险因素覆盖全面（不过度乐观或悲观）

## 关键原则

1. **趋势 > 绝对值** — 宏观数据的方向比当前水平更重要
2. **交叉验证** — 单一指标可能失真，多维度一致性提高置信度
3. **政策反身性** — 经济状态会触发政策响应，政策会改变经济轨迹
4. **避免精确的错误** — 宏观分析是概率判断，不追求点位预测
5. **保持更新** — 宏观状态快速演变，简报需要定期刷新

---

## 常见错误

❌ 只看单一指标（如仅看GDP）判断经济状态
❌ 忽视季节性和基数效应（同比数据可能失真）
❌ 将短期波动过度解读为趋势转折
❌ 忽视政策传导时滞（货币宽松到实体改善通常需2-3个季度）
❌ 混淆名义值和实际值（名义GDP含价格因素）
❌ 忽视数据修订（GDP等数据常发布初值后修订）

## 持续改进

完成宏观简报后，问自己：
1. 上一次简报的判断是否被验证？偏差原因是什么？
2. 哪些指标提供了最有价值的信号？
3. 是否有遗漏的重要维度或指标？
4. 简报的篇幅和深度是否适合受众需求？
5. 数据和观点是否有可追溯的记录？

最好的宏观分析通过持续跟踪和回测不断进化。建立数据档案、记录判断、验证结果，形成正向反馈循环。
